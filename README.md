# TeamFri-Front

### Dependencias a instalar

#### node modules
```
npm install
```

##### react-router-dom
```
npm install react-router-dom
```
##### sass
```
npm install sass
```
##### bootstrap
```
npm install bootstrap
```
##### reactstrap
```
npm install reactstrap
```
##### axios
```
npm install axios
```
##### JSON Server
```
npm install json-server
```
